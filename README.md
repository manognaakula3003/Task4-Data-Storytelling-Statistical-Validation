# Task 4 - Data Storytelling & Statistical Validation

## Objective

The objective of this project is to transform analytical findings into a compelling business story and validate key findings using statistical reasoning.

## Activities Performed

* Customer Analysis
* Product Analysis
* Country Analysis
* Monthly Sales Trend Analysis
* Deal Size Analysis
* Correlation Analysis
* Business Storytelling
* Statistical Validation

## Key Findings

* Euro Shopping Channel is the highest revenue-generating customer.
* Classic Cars generated the highest revenue.
* USA is the strongest market.
* November recorded the highest sales.
* Medium-sized deals contribute the largest share of revenue.
